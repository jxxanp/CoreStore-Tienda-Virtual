# CoreStore — Tienda Virtual 🛒

Aplicación de escritorio desarrollada en Java con JavaFX como proyecto final de Programación II en la Universidad de Córdoba.

## 📁 Contenido del repositorio

| Carpeta / Archivo | Descripción |
|---|---|
| `CoreStore - Imagenes de Referencia/` | Imágenes reales usadas como referencia visual para productos e interfaz |
| `CoreStore - Maqueta/` | Wireframes — esqueleto y estructura de cada pantalla |
| `CoreStore - Prototipo/` | Prototipo final de la interfaz con diseño visual aplicado de cada pantalla|
| `CoreStore - Inicio de Sesion y Catalogo.zip` | Módulo de inicio de sesión y catálogo |
| `CoreStore - Proyecto Final.zip` | Código fuente completo del proyecto |
| `Manual Usuario CoreStore.docx` | Manual de uso de la aplicación |

## 🎨 Proceso de diseño UI/UX

El proyecto documenta el flujo completo de diseño de interfaces:

**Referencia → Wireframe (Maqueta) → Prototipo final**

## ✨ Características

- Módulo de login y autenticación
- Catálogo de productos con búsqueda y filtros
- Carrito de compras
- Lista de deseos con animación de favoritos
- Panel administrativo para gestión de productos
- Historial de compras

## 🛠️ Tecnologías

- Java 17
- JavaFX SDK
- NetBeans IDE
- Estructuras de datos propias: Lista Doblemente Enlazada, ColaDeque
- Patrón de diseño: Singleton
